# Arquitetura ResiliênciaConectada

Este repositório contém a arquitetura da solução "ResiliênciaConectada", proposta para enfrentar eventos extremos da natureza, com foco em inundações e deslizamentos.

## Conteúdo
- `Desenho Tecnico/`: Contém os arquivos da arquitetura Archimate e o PDF exportado.

## Ferramentas
- Archi (para o desenho Archimate)
- Git Flow (para gerenciamento de versionamento)
